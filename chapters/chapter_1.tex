\chapter{Background \& Related Work}
\label{chp:background}

\section{Related Work}

GPS-denied navigation for drones has been approached with UWB, WiFi fingerprinting, and visual SLAM, which achieve high accuracy but impose computational or infrastructure costs that are prohibitive for weight-constrained micro-drones~\cite{}.

Visible-light positioning systems reach centimetre-level accuracy by combining many light anchors with known positions, but this density of lighting infrastructure is impractical for the environments considered in this thesis~\cite{}.

Approaches that rely on a single light source instead estimate only a bearing toward the source, or follow the light gradient with a single photodiode, which limits them to reaching one beacon in open space without obstacle avoidance~\cite{}.

This single-anchor limitation is the gap this thesis targets: it builds on the bearing and single-photodiode gradient methods of prior work by Harry Huang~\cite{}, and contributes their fusion, the use of all eight photodiodes for gradient estimation, and a spatial map that together enable multi-beacon and obstacle-aware navigation.

\section{Visible Light Communication \& Sensing}

Visible light communication uses modulated light sources to transmit data, enabling a light source to serve simultaneously as illumination and a detectable navigation beacon.

The intensity received from a single light source falls off with both distance and angle according to the Lambertian model, making a single intensity reading ambiguous and forcing conventional light-positioning systems to rely on multiple overlapping light sources~\cite{}.

Photodiodes convert incident light into a measurable electrical signal, and their directional sensitivity makes them suitable for estimating the angle of incidence of a light source.

\section{Bearing Angle Estimation}

Prior work introduced a bearing angle estimation algorithm that uses a ring of photodiodes to compute the relative angle toward a light source from the differential intensity readings across the array~\cite{}.

Array-based direction estimation is well established for radio signals such as Bluetooth, WiFi, and GPS, but remains comparatively underexplored for visible light~\cite{}.

The bearing angle provides a single scalar direction toward the light source but contains no spatial information about the light environment.

\section{Gradient-Based Source Seeking}

Gradient-based source seeking guides an agent toward the maximum of a scalar field, such as the intensity field around a light source.

Prior work followed this gradient using a single photodiode, inferring the direction of steepest increase from how the intensity changes as the drone moves~\cite{}.

Because a single sensor must infer the gradient from motion alone, this approach is slow and sensitive to drift, which motivates estimating the gradient from several sensors at once.

\section{Simulation Environment}

The simulation environment used in this thesis was developed in prior work and provides a physics-accurate Webots model of the drone platform with realistic light sensor simulation; for a full description the reader is referred to~\cite{}.

Simulation provides a reproducible, hardware-independent, and reasonably accurate environment in which the navigation algorithms can be developed and tested before deployment on hardware.
